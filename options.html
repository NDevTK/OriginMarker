How to Setup OriginMarker

To get started, OriginMarker needs you to designate a specific bookmark. The extension will then use this bookmark to display its unique visual markers based on the website you are currently viewing.

Steps:

  1. Choose or Create a Bookmark: You can either create a brand new bookmark or pick an existing one that you want OriginMarker to use. This bookmark will effectively become your "OriginMarker display".
  2. Edit the Bookmark's Name (Title):
    • To enable Automatic Mode (where OriginMarker shows a unique emoji for each website origin): Change the bookmark's name to exactly * (a single asterisk) and save it.
    • To enable Manual-Only Mode (where OriginMarker shows a generic marker unless you customize it for a specific site): Change the bookmark's name to exactly ** (two asterisks) and save it.
  3. Activation: Once you've saved the bookmark with * or ** as its name, OriginMarker will automatically detect it. From then on, this bookmark's title will change to show the marker for the current website's origin.

You can find this special bookmark in your bookmarks list or place it on your bookmarks bar for easy visibility. Its title will dynamically update as you browse different websites.

How to use

  • Rename the folder/marker on the origin (empty name to remove) or just use Automatic mode. Marker names are saved for that origin unless removed.
  • When you rename a marker it will not change the mode.

Automatic mode (ID: *)

It will be different emojis (randomly chosen) per origin than other users.

Emoji Markers: Please note that the emoji markers generated in automatic mode are for differentiating website origins only. They do not imply any security endorsement, verification, or status of the websites visited.

Asterisk (*) Suffix: Markers automatically generated by the extension will have an asterisk (*) at the end (e.g., "ðŸ˜€*"). This suffix indicates that the marker is system-managed. If you manually rename a marker to include an asterisk at the end, it will not be saved as a custom marker for the current origin.

Manual only (ID: **)

Markers will not get a automatic value.

Storage Area Warning: Using "Sync" storage is convenient for accessing your markers and settings across devices. However, please be aware that if your Google account is compromised, the unique 'salt' used for generating automatic markers could also be accessed. This might allow someone to identify which websites correspond to your emoji markers if they also have access to your bookmarks. Choosing "Local" storage keeps the salt on this device only, enhancing privacy in such scenarios, but your markers and settings will not be synced.

Marker bookmark ID and mode are in the Local area.

Extension will rename with a * suffix so they won't get treated as user changed.
